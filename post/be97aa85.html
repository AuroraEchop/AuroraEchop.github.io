  1. Vue：Vue 是一个渐进式 JavaScript 框架，用于构建用户界面。它易于上手，同时也能与现代工具链以及各种库和框架无缝结合。Vue 本身是一个核心库，只关注视图层，也便于与第三方库或既有项目整合。
  2. Vite：Vite 是一个由 Vue 作者尤雨溪开发的新型前端构建工具，它利用原生 ES 模块（ESM）的特性，实现了快速的开发服务器启动和热更新（HMR）。Vite 本身并不是一个框架，而是一个构建工具，它可以与 Vue、React 等多种框架配合使用。
  3. ES6：全称为ECMAScript 2015，是JavaScript语言的一个重要版本，于2015年正式发布。它是JavaScript语言的一个重大更新，引入了许多新特性和改进，使得JavaScript的语法更加简洁、功能更加强大，也更易于编写和维护代码。ES6的发布标志着JavaScript语言进入了一个新的发展阶段，为现代JavaScript开发奠定了基础。
  4. 状态提升：在 React 中，状态提升（Lifting State Up）是一种常见的模式，用于在组件树中共享状态。当多个组件需要访问和修改相同的数据时，可以将状态提升到它们的最近共同祖先组件中。这样，所有需要访问该状态的组件都可以通过 props 从祖先组件获取数据。
