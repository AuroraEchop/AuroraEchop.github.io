JSON

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，易于阅读和编写，同时也易于机器解析和生成。JSON 是基于文本的，使用 Unicode 编码，可以表示由属性-值对组成的数据对象，或由有序序列值组成的数组。

JSON 的数据结构

JSON 支持两种主要的数据结构：

  1. 对象：

    • 一个无序的键值对集合，用花括号 {} 包裹。键（属性名）必须是字符串，值可以是任何有效的 JSON 数据类型（字符串、数字、布尔值、数组、对象或 null）。

  2. 数组：

    • 一个有序的值集合，用方括号 [] 包裹。数组中的值可以是任何有效的 JSON 数据类型。

JSON 与 JavaScript 的关系

尽管 JSON 的名称中包含 “JavaScript”，但它是一种独立于语言的数据格式。JSON 的语法与 JavaScript 的对象字面量语法非常相似，但它们并不完全相同。以下是主要区别：

  1. 字符串必须用双引号：

    • JSON 中的字符串必须用双引号 " 包裹，不能使用单引号 '。

  2. 键（属性名）必须是字符串：

    • JSON 中的键必须是字符串，不能是其他类型（如数字或布尔值）。

  3. 不支持函数和 undefined：

    • JSON 不支持函数和 undefined。如果尝试序列化包含函数或 undefined 的对象，这些值会被忽略或转换为 null。

  4. 不支持循环引用：

    • JSON 不支持循环引用。如果尝试序列化包含循环引用的对象，会抛出错误。
