字符串方法问题

发表于2026-03-06|更新于2026-02-14|js
|总字数:1.3k|阅读时长:4分钟|浏览量:

常见考点

1. 基本字符串方法

  • charAt 和 charCodeAt 的区别是什么？如何使用它们获取字符串中某个字符或字符的 Unicode 编码？
    • 前面是用来获取字符串中某个下标的字符，返回字符，后者是获取某个位置字符的编码，返回编码
  • 如何通过 concat 拼接字符串？与 + 和模板字符串拼接相比有什么优缺点？
    • concat 拼写字符串可以一次传入多个字符串，实现链式链接，但是 +方便书写，而且可读性强，所见即所得。

2. 查找与判断

  • indexOf、lastIndexOf 和 includes 的区别是什么？这些方法在字符串查找中如何选择？
    • indexOf 是查找 子字符串 在字符串中的第一次出现的位置。返回下标
    • lastIndexOf(“a”) 查找 a 在字符串中最后出现的位置
  • 如何使用 startsWith 和 endsWith 判断字符串的开头或结尾内容？
    • 省

3. 提取子字符串

  • slice、substring 和 substr 的区别是什么？在什么情况下使用哪种方法？
    • slice 和 substring的参数都是绝对定位的， substr第二个参数（长度）是从第一个参数开始算起相对定位，返回字符串
    • slice支持负数 ，substring第二个参数可以小于第一个参数，自动反转
  • 你会如何使用 slice 实现字符串的反转？
    • 通过递归 然后使用slice(-1), 实际开发中使用split('').reverse().join('')

4. 大小写转换

  • 如何使用 toUpperCase 和 toLowerCase 实现字符串大小写转换？
  • 如何实现一个首字母大写的函数？比如将 "hello world" 转换为 "Hello World"？
    • str[0].toUpperCase() + str.substr(1,5) + str.[6].toUpperCase + str.substr(7)
    • 返回字符串

5. 字符串替换

  • replace 和 replaceAll 的区别是什么？在什么场景下需要使用 replaceAll？
    • 匹配一个和匹配所有，把字符串中第一个字符串替换成第二个字符串
  • replace 中的正则表达式如何实现全局替换？如何在替换中捕获匹配的内容并进行替换？
    • replace替换,match返回对象，包括下标,search返回下标这三个功能和 search 查找类似，但是 indeOf 不能传入正则

6. 去除空白字符

  • trim、trimStart 和 trimEnd 的区别是什么？如何用它们去除字符串开头strimStart或结尾的空格trimEnd？
  • 如何去除字符串中所有的空白字符？是否可以使用正则表达式实现？

7. 分割与连接

  • split 的用法是什么？如何使用它将字符串按特定分隔符转换为数组？
    • 使用正则表达式
  • 如何使用 split 和 join 方法实现字符串的重组，比如反转单词顺序等？

8. 重复与填充

  • repeat 方法的作用是什么？如何使用它生成一个重复的字符串？
    • repeat(num), num表示重复的次数。可以字符串转数字
  • padStart 和 padEnd 如何实现字符串的填充？请举例说明如何用它们补齐字符串的长度。

9. 模板字符串

  • 什么是模板字符串？如何通过模板字符串来插入变量和表达式？
  • 模板字符串中的 ${} 语法还能做什么操作？比如进行数学运算或调用函数。

10. 高级字符串操作

  • 如何使用正则表达式 match、matchAll、search 和 split 方法处理复杂文本？
  • 如何通过 replace 和回调函数来动态替换内容？比如替换字符串中的占位符。

11. Unicode 与编码

  • 如何判断一个字符串是否包含某个 Unicode 字符？如何使用 codePointAt 和 fromCodePoint 处理 Unicode 编码？
  • normalize 的作用是什么？在多语言字符处理中，它如何处理不同编码方式的字符？

12. 字符串比较

  • JavaScript 中字符串是如何比较的？localeCompare 的作用是什么？
  • 如何根据特定的语言或地区对字符串进行排序？localeCompare 如何实现多语言排序？

13. [[字符串操作练习题]]

  • 如何实现字符串的反转？
  • 如何判断一个字符串是否为回文？
  • 给定一个句子，如何将每个单词的首字母大写？
  • 如何提取字符串中的数字并进行求和？

14. 处理特殊字符

  • 如何处理字符串中的特殊字符，比如换行符、制表符等？如何用正则表达式匹配这些字符？
  • 在 HTML 中如何转义特殊字符？如 & 转为 &amp; 等？

15. 性能与优化

  • 如果有大量字符串拼接任务，使用 + 和 Array.join 拼接的性能有何不同？
  • 在字符串较长或操作频繁的情况下，是否有其他高效的字符串操作方法或工具可以推荐？
