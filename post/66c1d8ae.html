操作数组的方法

发表于2025-09-18|更新于2025-09-18|vue
|总字数:1.5k|阅读时长:6分钟|浏览量:

改变原数组的方法

这些方法会直接修改调用它们的原数组。

1. 添加/删除元素

  • push(): 在数组末尾添加一个或多个元素，并返回新数组的长度。

  • pop(): 删除数组末尾的最后一个元素，并返回该元素。

  • unshift(): 在数组开头添加一个或多个元素，并返回新数组的长度。

  • shift(): 删除数组开头的第一个元素，并返回该元素。

    JavaScript

  • splice(start, deleteCount, item1, item2, ...): 万能方法，可以添加、删除或替换数组中的元素。

    • start: 指定修改的开始位置。
    • deleteCount: 要删除的元素数量。
    • item1, item2, ...: 要添加的新元素。

    JavaScript

2. 排序/反转

  • sort(compareFunction): 对数组进行排序，默认按字符串Unicode编码排序。可以传入一个比较函数来自定义排序规则。

    JavaScript

  • reverse(): 反转数组中元素的顺序。

    JavaScript

3. 填充

  • fill(value, start, end): 用一个固定值填充数组中指定范围内的所有元素。

    JavaScript

不改变原数组的方法

这些方法会返回一个新数组，而不会修改原数组。

1. 提取/连接

  • slice(start, end): 返回一个从 start 到 end（不包括 end）的浅拷贝的新数组。

    JavaScript

  • concat(arr1, arr2, ...): 连接两个或多个数组，并返回一个新数组。

    JavaScript

  • join(separator): 将数组的所有元素连接成一个字符串。

    JavaScript

2. 查找/判断

  • indexOf(element, fromIndex): 返回指定元素在数组中第一次出现的索引，如果不存在则返回 -1。

    JavaScript

  • lastIndexOf(element, fromIndex): 返回指定元素在数组中最后一次出现的索引，如果不存在则返回 -1。

    JavaScript

  • includes(element, fromIndex): 判断数组是否包含某个元素，返回布尔值。

    JavaScript

  • find(callbackFunction): 返回数组中第一个满足条件的元素。

    JavaScript

  • findIndex(callbackFunction): 返回数组中第一个满足条件的元素的索引。

    JavaScript

  • every(callbackFunction): 测试数组中的所有元素是否都通过了指定函数的测试，返回布尔值。

    JavaScript

  • some(callbackFunction): 测试数组中是否至少有一个元素通过了指定函数的测试，返回布尔值。

    JavaScript

3. 遍历/转换

  • forEach(callbackFunction): 对数组的每个元素执行一次提供的函数，没有返回值。

    JavaScript

  • map(callbackFunction): 对数组的每个元素执行一次提供的函数，并返回一个新数组，包含调用函数的结果。

    JavaScript

  • filter(callbackFunction): 创建一个新数组，其中包含通过指定函数测试的所有元素。

    JavaScript

  • reduce(callbackFunction, initialValue): 对数组中的所有元素执行一个累加器函数，并返回一个单一的值。

    JavaScript

  • reduceRight(callbackFunction, initialValue): 与 reduce 类似，但从数组的右侧开始处理。

2. 数组扁平化

  • flat(depth) (ES2019): 创建一个新数组，其中所有子数组元素递归地连接到指定深度。

    JavaScript

  • 如果不支持 flat，可以通过 reduce 或递归实现。

总结

选择哪种方法取决于你的需求：

  • 是否需要改变原数组？ 如果需要，请使用 push, pop, shift, unshift, splice, sort, reverse, fill 等。
  • 是否需要返回一个新数组？ 如果需要，请使用 slice, concat, map, filter, reduce, flat 等。
  • 你想要进行什么操作？ 添加、删除、查找、遍历、排序、转换等等。

熟练掌握这些方法将大大提高你在 JavaScript 中处理数组的效率和灵活性。你对哪种数组操作更感兴趣，或者有什么具体的需求吗？
