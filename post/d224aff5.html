filter函数

发表于2025-09-18|更新于2025-09-18
|总字数:375|阅读时长:1分钟|浏览量:

filter 是 JavaScript 中数组的一个常用方法，用于创建一个新数组，新数组中的元素是通过测试函数（即回调函数）筛选出来的。filter 方法不会修改原数组，而是返回一个包含符合条件的元素的新数组。

基本语法

[!NOTE] 总结

  1. filter前 - 的是操作的 数组 array
  2. filter后 - 括号是测试函数，用来设置设置条件。 测试函数返回值一定是 布尔类型
  3. filter 默认将数组每一项传递给 回调函数 的参数 element
  4. 使用箭头函数作为回调函数：
    • 适合：与this无关的回调 （定时器、数组方法回调）
    • 不适和：this有关（事件回调、对象方法）

filter 方法的基本语法如下：

例子：

  • array：要处理的数组。
  • function：测试函数，用于决定是否保留当前元素。
    • element：当前正在处理的元素。
    • index（可选）：当前正在处理的元素的索引。
    • array（可选）：调用 filter 的数组。
  • thisArg（可选）：执行回调时使用的 this 值。
  • newArray：返回的新数组，包含通过测试函数筛选出来的元素。

返回值

filter 方法返回一个新数组，包含通过测试函数筛选出来的元素。如果没有任何元素通过测试函数，返回一个空数组。
