CDN
发表于2026-03-06|工程化
CDN 全称是 Content Delivery Network（内容分发网络） 一、 什么是 CDN？（通俗版）场景： 假设你在深圳，你想买一瓶北京厂家生产的可乐。 没有 CDN 的情况（直连源站）： 你下单 -> 厂家从北京仓库发货 -> 跨越几千公里 -> 送到深圳。 有 CDN 的情况（边缘节点）： 厂家提前把可乐运送到全国各地的分仓库（CDN 节点）。 你下单 -> 系统发现你在深圳 -> 自动指派深圳南山区的分仓库给你发货 -> 1小时送到。 技术总结： CDN 就是建立在网络之上的“智能虚拟网络”。它在全世界各处都放了服务器（节点），当用户请求资源（图片、JS、CSS）时，CDN 会让用户去离他最近、最快的那个服务器下载，而不是大家都去挤那台中心服务器。
